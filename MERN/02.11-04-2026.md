## Setting of Keycloak in Local Machine 

**Step 1:** Need to setup Rancher/ Docker Desktop.

**Step 2:** Though we are using Docker compose make sure to collect the docker-compose.yml file from google or use the below github link

https://github.com/nirmalakumarsahu/docker-compose-files/blob/master/Keycloak%20with%20Postgres%20SQL/docker-compose.yml

**Note:** docker-compose.yml / yaml  is file where we can list down the service we need to install.

**Step 3:** Do the changes accordingly in side the yml file like DB Name, username, password, keycloak details. 

**Step 4:** Open the terminal or CMD where your docker-compose file is present.

**Step 5:** Use the below command to create and run the container.

**Note:** Make sure your Rancher/ Docker desktop should on before running then below command

```shell
docker-compose up -d
```

Step 6: Now you can go to your Rancher/ Docker Desktop and check. After that use the below URL to access the Keycloak.

http://localhost:8989
